-------------------------------------------------------------Dot Matrix Display----------------------------------------------------------
-----------------------------------------------------------------------------------------------------------------------------------------
Project Description:
	>> Uses SIPO shift register 74HC595N 
	>> Every versions imporoved the project by one step 

-----------------------------------------------------------Version Description-----------------------------------------------------------
-----------------------------------------------------------------------------------------------------------------------------------------
Version 1
	• A simple 5*8 led dot matrix 
	• A sketch code for displaying some words : shows a basic word display and scroll 
	• Uses a single 74HC595 to control column while a row is handled by arduino uno output pins 
	• Arduino UNO or Arduino nano can be used 

Version 2 
	• 8*16 led dot matrix is build 
	• A custom hexadecimal english letter format is designed 
	• A full sentence can be scroll 
	• Total 3 74HC595N shift registers are used 
	• 8 BC547 transistors along with 1k resitors are used for switch and pull down resistors 
	
Version 3 
	• Upgrading version 2 
	• More text display styles will be added 
	
Version 4 
	• Bluetooth control is to be added 
Version 5 
	• Upgradition to Module free hardware
